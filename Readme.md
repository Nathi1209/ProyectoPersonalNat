# Enciclopedia Animales de Costa Rica

Proyecto personal desarrollado con **Vue 3 + Vite**.

## Framework elegido
**Vue 3** — Composition API con Single File Components (.vue)

## Opción elegida
**Opción 4: Enciclopedia Temática Interactiva**

## Descripción
Enciclopedia interactiva de fauna silvestre de Costa Rica. Permite explorar 10 animales con información detallada, búsqueda en tiempo real, filtros por categoría (Aves, Mamíferos, Reptiles, Anfibios), narración de audio y modo oscuro.

## Características
- Búsqueda en tiempo real por nombre, nombre científico o región
- Filtros por categoría de animal
- Modal de detalle con descripción, hábitat y estado de conservación
- Narración de audio para 3 animales
- Modo oscuro / claro
- Diseño responsivo

## Tecnologías
- Vue 3 (Composition API)
- Vite 6
- JavaScript (ES Modules)

## Cómo ejecutar
```bash
npm install
npm run dev
```
